Why Programming?

Ever since I was little I've always had an interest in anything technology related. I grew up playing Halo on the Xbox, then eventually moved to the PC where I started to play Minecraft like a lot of other people. It was during this time that I realized that hosting servers and programming were absolutely fascinating to me. Being able to create something and then share it with the world, albeit a little nerve racking for me, is something that I really wanted to do. I started small in Minecraft, making a pretty basic plugin called Ore-Logger (which I never released). Minecraft is the main reason why I consider Java to be my main programming language. Although I've dabbled in a few others including Web (HTML/CSS/JS/PHP), Python, C++, and C#, Java has always been what peaked my interest the most.
